Conectar ao Google Sheets

Sem API Key! Este método usa a exportação pública do seu Sheets. Siga os dois passos abaixo.

Como configurar (2 minutos)
1
Abra sua planilha no Google Sheets → clique em Compartilhar (canto superior direito) → em "Acesso geral" selecione "Qualquer pessoa com o link" como Leitor → Concluído.
2
Copie o ID da sua planilha da URL. É o trecho entre /d/ e /edit.
Ex: docs.google.com/spreadsheets/d/1hoBLtKJ...AvteA/edit
3
Cole o ID abaixo e clique em Conectar. Sem API Key, sem configuração extra.
